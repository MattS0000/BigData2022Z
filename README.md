# BigData2022Z
Project for Big Data.
In hdfs create project directory under ./user, ex. using `hdfs dfs -mkdir /user/project`.
Requires tree structure as follows:

    /user/project/
    
    ├── input
    
    │   ├── ...
    
    │   ├── crashes
    
    │   └── violations
    
    └── data
    
        ├── ...
        
        ├── crashes
        
        └── violations
